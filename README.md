# dither

![Preview](image.png)

A shadcn component for reactive dithered images.

## Installation

```bash
bunx --bun shadcn add https://dither.leomosley.com/r/dither.json
```

## Usage

```tsx
import { Dither } from "@/components/dither";

<Dither src="/your-image.png" />;
```
